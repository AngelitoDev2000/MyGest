SISTEMA GESTOR
MYGEST

MYGEST fue creado con el proposito de ayudar a las pequeñas empresas hotelereas que no cuentan con un sistema administrador computacional para poder gestionar optimamente sus hoteles.

MYGEST optimiza el sistema de reservas, almacenando la informacion en una segura base de datos, dando fiabilidad al usuario y facilitando las consultas y registros de gran informacion.
